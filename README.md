# poker-bego
Poker Bego (Go Fish) adalah game kartu (2-6 pemain) bertujuan mengumpulkan set 4 kartu berangka sama untuk poin.  Pemain bergantian meminta kartu spesifik (angka &amp; simbol) yang sudah dimiliki kepada lawan. Jika benar, ia dapat kartu dan lanjut. Jika salah, ia ambil 1 kartu dari tumpukan dan giliran selesai. Pemilik poin terbanyak menang.
